## NodeJS API template
A simple template for NodeJS API, using Nodemon to hot reload the application, setting up the API routes to another file and some others helpfull things. 


## Running The Server

1. Build the server
# install node modules and dependencies   
~~~
npm install
~~~

2. Run the server
# using Hot Reload  
~~~
npm run dev
~~~

or

# Starting the server
~~~
npm start
~~~
